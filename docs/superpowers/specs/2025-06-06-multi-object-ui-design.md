## 3. 动画调整

### 3.1 入场动画

| 属性 | 当前值 | 新值 | 说明 |
|------|--------|------|------|
| 缓动曲线 | `elasticOut` | `easeOutBack` | 轻微回弹，更优雅不夸张 |
| 时长 | 600ms | 450ms | 更快，更利落 |
| Stagger 延迟 | 120ms | 80ms | 更紧凑的节奏感 |

### 3.2 锚点脉冲（新增）

每个气泡弹出的同时，对应锚点同步做一次脉冲动画：
- 缩放：1.0 → 1.5 → 1.0
- 透明度：1.0 → 0.5 → 1.0
- 时长：300ms
- 缓动：`easeOutCubic`

实现方式：使用 `AnimationController` 的 `value` 驱动，在 `adjustedValue` 首次从 0 变为 >0 时触发一次脉冲。
